Hangman
by Matthew Magardino

to run (in Apple terminal):
  - download zip to Desktop
  - double click to unzip
  - rename folder to 'hangman-py'
  - type 'cd Desktop/hangman-py' and hit enter
  - write 'python3 hangman.py' and hit enter
  - enjoy :)


or:
  - double click executable (hangman)

requirements:
  - python3 installed
